# Página de Checkout (Repositório Recuperado)

Este projeto fazia parte do meu portfólio na [**conta antiga do GitHub**](https://github.com/ruthpaulino), à qual perdi acesso.  
Mantive o código aqui para preservar meu histórico e facilitar a visualização das minhas habilidades.

## Sobre o projeto
- Desenvolvido originalmente em 2023 para um desafio que participei em um processo seletivo.
- Tecnologias utilizadas: HTML, CSS, JS.
- Objetivo: Criar uma página de checkout de loja contendo: Campos obrigatórios com validação, acessibilidade, responsividade e um pouco de JavaScript para calcular o valor do produto.
